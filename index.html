Re.Edit

CodeMirror based web editor. It is ment to be used like a normal editor sort of like a vi or gedit, except that every thing is edited on the server. Every thing is stored on the server if you leave or come back, every thing is just like how it was.

Requirements

  • nodejs
  • express
  • CodeMirror
  • sass
